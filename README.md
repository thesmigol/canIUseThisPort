# Can I use this port?

I always need to check ports... so i make it

# Install

npm i can-i-use-this-port --save

## Usage

    const  ciutp =  require("can-i-use-this-port");
    
	cipt(port); -> return the next free port to use
	cipt(3000); 
	{
		return -> 3001 (the port will be check before it)
	}
